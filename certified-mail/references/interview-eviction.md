# 인터뷰: 명도·인도 청구 내용증명

## Step E1. 발신인·수신인
```
발신인: 소유자/임대인 — 성함/주소/연락처
수신인: 점유자/임차인 — 성함/주소/연락처
```

## Step E2. 부동산 정보
```
1) 주소
2) 종류: 주거·상가·토지 등
3) 면적 (알고 있는 경우)
```

## Step E3. 명도 요구 사유
```
왜 비워달라고 요구하시나요?

1) 임대차 계약 기간 만료
   → 만료일: YYYY년 M월 D일
   → 갱신 통보 여부

2) 계약 해지 (차임 연체 등)
   → 연체 기간 및 금액

3) 소유권 이전 (매각 등)

4) 기타
```

## Step E4. 이행 기한
```
며칠 이내에 비워줄 것을 요구하시겠어요?

권장: 도달 후 30일 이내
직접 지정: ___일 이내
```

## Step E5. 미이행 시 조치
```
1) 명도소송 제기
2) 부동산 인도 강제집행 신청
3) 손해배상 청구 (점유로 인한 차임 상당액)
```
