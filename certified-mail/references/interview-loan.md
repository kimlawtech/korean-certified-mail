# 인터뷰: 대여금 반환 청구 내용증명

## Step L1. 발신인 정보
```
빌려준 사람(채권자) 정보:
1) 성함 / 2) 주소 / 3) 연락처
```

## Step L2. 수신인 정보
```
빌린 사람(채무자) 정보:
1) 성함 / 2) 주소 / 3) 연락처
```

## Step L3. 대여 내역
```
1) 대여 날짜: YYYY년 M월 D일
2) 대여 금액: 금 X,XXX만원
3) 대여 방법: 계좌이체 / 현금
   (계좌이체 시 이체 확인증 보관 권장)
4) 차용증·각서 있음/없음
5) 약정 변제일: YYYY년 M월 D일 (없으면 "약정 없음")
```

## Step L4. 상환 이력
```
일부라도 받은 적 있나요?
1) 있음 → 날짜·금액
2) 없음

이미 반환 요청한 적 있나요?
1) 있음 → 방식·상대방 반응
2) 없음
```

## Step L5. 이자 약정
```
이자를 약정했나요?
1) 있음 → 연 ___% 또는 월 ___%
2) 없음 (법정이율 연 5% 적용 가능)

이자는 약정한 내용만 청구 가능합니다.
```

## Step L6. 이행 기한
```
몇 일 이내 반환 요구: 권장 14일 이내
```

## Step SAVE
```
파일 저장 폴더 (기본값: ~/Desktop)
```
