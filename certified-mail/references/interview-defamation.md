# 인터뷰: 명예훼손·사생활 침해 내용증명

## Step DM1. 발신인·수신인
```
발신인: 피해자 — 성함/주소/연락처
수신인: 가해자 — 성함(또는 온라인 닉네임)/주소/연락처
  (주소 불명인 경우: 가해자가 활동하는 플랫폼·채널 주소를 대체 기재)
```

## Step DM2. 침해 내용
```
어떤 내용이 유포되었나요?

1) 게시 날짜: YYYY년 M월 D일 (또는 최초 발견일)
2) 게시 위치: 플랫폼·URL (예: 유튜브, 인스타그램, 온라인 커뮤니티)
3) 내용 요약: (구체적 문구를 직접 입력해도 됩니다)
4) 사실 여부: 허위 사실 / 사실이나 무단 공개
5) 스크린샷·캡처 저장 여부
```

## Step DM3. 피해 현황
```
어떤 피해가 발생했나요?

1) 직장·사업상 피해
2) 사회적 평판 훼손
3) 정신적 피해
4) 기타
```

## Step DM4. 요구사항
```
1) 즉시 해당 게시물·콘텐츠 삭제
2) 추가 유포 중단
3) 공개 사과
4) 손해배상: 금 XXX만원 (위자료 포함)
```

## Step DM5. 이행 기한
```
삭제 기한 권장: 내용증명 도달 즉시 또는 3일 이내
손해배상 이행 기한: 14일 이내
```

## Step DM6. 미이행 시 조치
```
1) 민사소송 (손해배상 청구)
2) 형사 고소 (형법 §307, 정보통신망법 §70)
3) 플랫폼 신고 및 방통위 신고
```

## Step SAVE
```
파일 저장 폴더 (기본값: ~/Desktop)

※ 주의: 허위 사실이 아닌 경우 명예훼손이 성립하지 않을 수 있습니다.
  게시 내용이 공공의 이익에 관한 사실인 경우 위법성이 조각될 수 있습니다.
  형사 고소 전 변호사 상담을 권장합니다.
```
