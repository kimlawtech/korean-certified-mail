# 인터뷰: 서비스 이행 청구 내용증명

## Step S1. 발신인·수신인
```
발신인: 서비스를 의뢰한 쪽 — 성함/주소/연락처
수신인: 서비스 제공자 — 이름·상호/주소/연락처
```

## Step S2. 계약 내용
```
1) 서비스 종류 (예: 웹 개발, 디자인, 교육, 이사 등)
2) 계약일
3) 계약 금액
4) 약정 완료일 또는 납기일
5) 선급금 지급 여부: 금 XXX만원 지급 / 없음
```

## Step S3. 불이행 현황
```
어떤 부분이 이행되지 않았나요?
1) 납기 지연 — 예정일: YYYY년 M월 D일, 현재까지 미완료
2) 불완전 이행 — 일부만 이행
3) 이행 거부

이미 이행 요청한 적 있나요? (방식·날짜·반응)
```

## Step S4. 요구사항
```
1) 기한 내 이행 완료
2) 선급금 반환 (이행 불가 시): 금 XXX만원
3) 손해배상: 금 XXX만원
```

## Step S5. 이행 기한
```
권장: 도달 후 7~14일 이내
```

## Step SAVE
```
파일 저장 폴더 (기본값: ~/Desktop)
```
