# 인터뷰: 저작권 침해 경고 내용증명

## Step C1. 발신인·수신인
```
발신인: 저작권자 — 성함·상호/주소/연락처
수신인: 침해자 — 성함·상호/주소/연락처
  (특정 불가 시: 플랫폼·채널 운영자 주소 기재)
```

## Step C2. 저작물 정보
```
1) 저작물 종류: 사진·그림·글·음악·영상·소프트웨어·기타
2) 저작물 제목 또는 설명
3) 창작·공표일 (알고 있는 경우)
4) 저작권 등록 여부 (한국저작권위원회 등록 시 등록번호)
5) 원본 소재지 (URL 또는 발행물)
```

## Step C3. 침해 내용
```
1) 침해 발견일: YYYY년 M월 D일
2) 침해 위치: URL / 플랫폼 / 오프라인 매체
3) 침해 방식: 무단 복제·게시 / 무단 편집 / 무단 상업적 이용 / 기타
4) 현재 침해 지속 여부
5) 스크린샷·증거 캡처 여부
```

## Step C4. 요구사항
```
1) 즉시 사용 중단 및 삭제
2) 해당 플랫폼 내 전체 게시물 삭제
3) 손해배상: 금 XXX만원
4) 서면 사과
```

## Step C5. 이행 기한
```
삭제·중단: 내용증명 도달 즉시
손해배상: 14일 이내
```

## Step C6. 미이행 시 조치
```
1) 저작권법 §123·§125에 따른 민사 소송
2) 한국저작권위원회 조정 신청
3) 저작권 침해 형사 고소
```

## Step SAVE
```
파일 저장 폴더 (기본값: ~/Desktop)
```
