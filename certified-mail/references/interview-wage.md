# 인터뷰: 임금·급여 체불 내용증명

## 적용 상황

- 퇴직 후 임금·퇴직금 미지급
- 재직 중 임금 일부 미지급
- 주휴수당·연장근로수당 미지급
- 최저임금 위반

---

## Step W1. 발신인 정보

```
먼저 보내는 분 정보를 알려주세요.

1) 성함
2) 주소 (등기우편 발송 기준)
3) 연락처 (전화번호)

예) 홍길동 / 서울시 강남구 테헤란로 123 / 010-1234-5678
```

## Step W2. 수신인 정보

```
임금을 지급하지 않은 고용주(사업장) 정보를 알려주세요.

1) 사업장 이름
2) 대표자 성함
3) 사업장 주소 (내용증명 발송 주소)
4) 연락처 (알고 있는 경우)

주소를 모르는 경우: 사업자등록증이나 간판에 적힌 주소를 사용하세요.
```

## Step W3. 근무 기간

```
언제부터 언제까지 일하셨나요?

예) 2025년 3월 1일 ~ 2025년 12월 31일 (퇴직일 포함)

퇴직하지 않은 경우: "재직 중"으로 표시합니다.
```

## Step W4. 미지급 항목

```
어떤 돈을 못 받으셨나요? (해당 항목 모두 선택)

1) 월급 (기본임금)
2) 퇴직금
3) 주휴수당
4) 연장근로수당 (야근·초과근무)
5) 연차미사용수당
6) 기타 수당 (직접 입력)

여러 개 해당되면 모두 말씀해 주세요.
```

## Step W5. 미지급 기간 및 금액

```
각 항목별로 알려주세요.

예시)
- 월급: 2025년 11월분, 12월분 미지급 — 각 250만원, 합계 500만원
- 퇴직금: 퇴직일 기준 300만원 미지급
- 연장수당: 2025년 10월~12월 월 20시간 초과 — 약 150만원

정확한 금액을 모르는 경우 근사치로 입력하시고,
"정확한 금액은 추후 확인 예정"으로 처리할 수 있습니다.
```

## Step W6. 기존 요구 이력

```
이미 사업주에게 임금 지급을 요청하신 적 있나요?

1) 있음 — 언제, 어떻게 요청했나요? (문자·전화·대면 등)
   → 사업주 반응은? (무시·거절·부분 지급 등)
2) 없음

요청 기록(문자·카카오톡·이메일 등)이 있으면 증거로 활용할 수 있습니다.
```

## Step W7. 이행 기한

```
몇 일 안에 지급해 달라고 요구하시겠어요?

기본 설정: 내용증명 도달 후 7일 이내 (권장)
직접 지정: ___일 이내

근로기준법 §36은 퇴직 후 14일 이내 지급 의무를 규정합니다.
```

## Step W8. 후속 조치 예고

```
미지급 시 어떤 조치를 예고하시겠어요? (선택)

1) 고용노동부 임금체불 진정 (기본 권장)
2) 민사소송 (지급명령 또는 소액심판)
3) 형사 고소 (근로기준법 §109)
4) 복수 선택 가능

고용노동부 진정은 무료이며 빠른 처리가 가능합니다.
형사 고소 예고 문구 포함 시 사업주에게 더 강한 압박이 됩니다.
```

## Step SAVE

```
파일을 어디에 저장할까요?

기본값: ~/Desktop
직접 지정: 폴더 경로를 입력해 주세요.
예) /Users/홍길동/Documents/법률서류
```

## 생성되는 내용증명 구조

```
※ 면책 문구

내용증명서 (임금청구)

발신인: [성명] / [주소] / [연락처]
수신인: [사업장명] 대표 [대표자명] / [주소]
작성일: YYYY년 M월 D일

제목: 미지급 임금 지급 청구 내용증명

1. 당사자 관계
   발신인은 YYYY년 M월 D일부터 YYYY년 M월 D일까지
   귀사에서 [직종/직무]로 근무한 근로자입니다.

2. 미지급 임금 내역
   [항목별 금액 및 기간]
   합계: 금 X,XXX,XXX원

3. 법적 근거
   근로기준법 제[조]에 따라...

4. 요구사항
   이 내용증명 도달일로부터 7일 이내에
   위 금원을 발신인 명의 계좌로 지급하여 주시기 바랍니다.

5. 미이행 시 조치
   위 기한까지 이행하지 않으실 경우 고용노동부 임금체불 진정,
   민사소송 제기 등 법적 조치를 취할 것임을 알려드립니다.

   [발신인 서명]
```
