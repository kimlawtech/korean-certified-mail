# 인터뷰: 임대차 갱신거절 통보 내용증명

## 주의사항 (인터뷰 시작 전 안내)
```
갱신거절 통보는 통지 기한이 있습니다.

주거용: 임대차 만료 6개월 전 ~ 2개월 전 사이 통지 필수
상가용: 임대차 만료 6개월 전 ~ 1개월 전 사이 통지 필수

이 기간을 놓치면 묵시적 갱신이 성립해 거절이 불가능합니다.
현재 만료일 기준으로 통지 가능 여부를 먼저 확인합니다.

임대차 만료일이 언제인가요?
```
→ 만료일 입력 후 오늘(스킬 실행 시점의 날짜) 기준으로 통지 가능 여부 자동 판단한다.
→ 기간 경과 시 경고: "통지 가능 기간이 지났습니다. 갱신거절이 불가능할 수 있습니다."

## Step R1. 발신인·수신인
```
발신인: 임대인 — 성함/주소/연락처
수신인: 임차인 — 성함/주소/연락처
```

## Step R2. 부동산 및 계약 정보
```
1) 부동산 주소
2) 종류: 주거·상가
3) 현재 계약 만료일
4) 보증금·월세 금액
```

## Step R3. 거절 사유
```
왜 갱신을 거절하시나요?

주택임대차보호법 §6의3에서 정한 거절 사유 (해당 항목 선택):
1) 임대인 또는 직계존비속이 직접 거주할 예정
2) 임차인이 2기 이상 차임을 연체
3) 임차인이 임대인 동의 없이 전대
4) 임차인이 주택의 일부를 파손
5) 계약 기간 중 상호 합의로 계약 해지한 경우
6) 재건축·철거 계획 (관련 허가 확인 필요)

※ 단순 임대인의 거절 의사만으로는 갱신 거절 불가
   위 사유 중 하나 이상이 있어야 합니다.
```

## Step R4. 이행 요청
```
임차인에게 추가로 요청할 사항:
1) 만료일까지 퇴거 요청
2) 보증금 반환 일정 협의 희망
```
