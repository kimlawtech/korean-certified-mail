# 인터뷰: 하자보수 청구 내용증명

## Step DF1. 발신인·수신인
```
발신인: 성함/주소/연락처
수신인: 시공사·판매자 이름/주소/연락처
```

## Step DF2. 계약 정보
```
1) 계약 종류: 공사도급·매매·리모델링 등
2) 계약일
3) 목적물 인도(준공) 날짜
4) 계약 금액
```

## Step DF3. 하자 내용
```
어떤 하자가 발생했나요?

1) 발견 날짜: YYYY년 M월 D일
2) 하자 위치 (예: 거실 천장, 외벽 등)
3) 하자 내용 (예: 누수, 균열, 기울어짐, 제품 결함 등)
4) 사진·영상 촬영 여부

복수 하자인 경우 항목별로 나열해 주세요.
```

## Step DF4. 기존 요구 이력
```
하자 보수를 이미 요청한 적 있나요?
1) 있음 → 날짜·방식·상대방 반응
2) 없음
```

## Step DF5. 요구사항
```
1) 하자 보수 시공 요구
2) 보수 불가 시 손해배상 금액: 금 XXX만원
3) 계약 대금 감액 청구
4) 복수 선택 가능
```

## Step DF6. 이행 기한
```
권장: 도달 후 14~30일 이내
```

## Step SAVE
```
파일 저장 폴더 (기본값: ~/Desktop)

※ 주의: 매매 하자 제척기간 6개월, 도급 하자 1년/5년
   하자 발견일로부터 시간이 많이 지났다면 소멸시효를 먼저 확인하세요.
```
