# 인터뷰: 계약 이행 촉구 내용증명

## Step P1. 발신인 정보
```
성함 또는 회사명·대표자 / 주소 / 연락처
```

## Step P2. 수신인 정보
```
성함 또는 회사명·대표자 / 주소 / 연락처
```

## Step P3. 계약 내용
```
1) 계약 종류 및 날짜
2) 상대방이 이행해야 할 내용
3) 약정 이행 기한
4) 계약서 있음/없음
```

## Step P4. 불이행 현황
```
1) 어떤 부분이 이행되지 않았나요?
2) 기한: YYYY년 M월 D일 초과 여부
3) 이미 이행 요청한 적 있나요? (방식·날짜·반응)
```

## Step P5. 이행 촉구 기한
```
이번 내용증명 도달 후 며칠 이내로 이행을 요구하시겠어요?

권장: 7~14일
직접 지정: ___일 이내

기한 내 미이행 시 → 계약 해제권 발생 (민법 §544)
이 점을 내용증명에 명시하시겠어요? (예/아니오)
```

## Step P6. 미이행 시 조치
```
미이행 시 어떤 조치를 예고하시겠어요?
1) 계약 해제 통보
2) 손해배상 청구
3) 민사소송 제기
4) 복수 선택 가능
```
