# 인터뷰: 보증금 반환 청구 내용증명

## 적용 상황

- 임대차 계약 종료 후 보증금 미반환
- 전세·월세 보증금 일부만 돌려줌
- 임대인이 부당 공제 주장

---

## Step D1. 발신인 정보

```
보내는 분(임차인) 정보를 알려주세요.

1) 성함
2) 현재 주소 (내용증명 발송 기준 — 이미 이사했으면 새 주소)
3) 연락처
```

## Step D2. 수신인 정보

```
임대인 정보를 알려주세요.

1) 임대인 성함 (또는 법인명)
2) 임대인 주소 (등기부등본에 기재된 주소가 정확합니다)
3) 연락처 (알고 있는 경우)
```

## Step D3. 임대차 목적물

```
어떤 부동산의 보증금인가요?

1) 주소 (예: 서울시 마포구 XX동 XX호)
2) 종류: 주거용(아파트·빌라·오피스텔) / 상가·사무실
3) 보증금 금액: 금 X,XXX만원
4) 월세 금액 (해당 시)
```

## Step D4. 계약 기간

```
임대차 계약 기간을 알려주세요.

1) 계약 시작일: YYYY년 M월 D일
2) 계약 만료일: YYYY년 M월 D일
3) 실제 퇴거일 (이미 이사한 경우): YYYY년 M월 D일

아직 퇴거 전인 경우: "퇴거 예정일"을 입력해 주세요.
```

## Step D5. 반환 요구 이력

```
임대인에게 보증금 반환을 요구한 적 있나요?

1) 있음 — 언제, 어떤 방식으로?
   → 임대인 반응은? (묵묵부답·부분 반환·공제 주장 등)
2) 없음

임대인이 공제를 주장한 경우: 어떤 항목을 공제한다고 했나요?
(수리비·청소비·연체 차임 등)
```

## Step D6. 이행 기한

```
며칠 이내 반환을 요구하시겠어요?

권장: 내용증명 도달 후 7일 이내
직접 지정: ___일 이내

퇴거 시 열쇠·카드 반납과 동시에 반환 요구하는 것이 일반적입니다.
```

## Step D7. 계좌 정보

```
보증금을 받을 계좌를 알려주세요.

은행명: 
계좌번호: 
예금주: 

(MCP 연결된 경우 로컬 처리 — Claude에 노출되지 않습니다)
```

## Step SAVE

```
파일 저장 폴더를 알려주세요. (기본값: ~/Desktop)
```

---

## 생성되는 내용증명 구조

```
내용증명서 (보증금 반환 청구)

발신인 / 수신인 / 작성일

1. 당사자 관계 — 임대차 계약 개요
2. 보증금 반환 청구 내역
   - 계약 보증금: X,XXX만원
   - 반환받은 금액: X,XXX만원
   - 청구 금액: X,XXX만원
3. 법적 근거 — 민법 §654, 주택임대차보호법 §3의2 (해당 시)
4. 요구사항 — X일 이내 반환 + 입금 계좌
5. 미이행 시 조치 — 임차권등기명령 신청, 민사소송 제기
```
