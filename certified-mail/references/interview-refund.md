# 인터뷰: 환불·청약철회 청구 내용증명

## Step RF1. 발신인·수신인
```
발신인: 소비자 — 성함/주소/연락처
수신인: 판매자·사업자 — 상호/대표자/주소/연락처
```

## Step RF2. 거래 내역
```
1) 구매 날짜: YYYY년 M월 D일
2) 상품·서비스명
3) 구매 금액: 금 XXX만원
4) 구매 경로: 온라인(쇼핑몰·앱) / 오프라인 / 전화
5) 주문번호 또는 영수증 번호 (있는 경우)
```

## Step RF3. 반품·환불 사유
```
환불을 원하는 이유는 무엇인가요?

1) 단순 변심 (수령 후 7일 이내만 가능)
2) 제품 불량·하자
3) 광고와 다른 상품
4) 배송 지연
5) 서비스 미이행

※ 단순 변심은 수령일로부터 7일 이내 청약철회가 가능합니다 (전자상거래법 §17).
  오늘 날짜 기준으로 7일 이내인가요?
```

## Step RF4. 기존 환불 요구 이력
```
판매자에게 이미 환불 요청한 적 있나요?
1) 있음 → 날짜·방식·판매자 반응
2) 없음
```

## Step RF5. 요구사항
```
1) 전액 환불: 금 XXX만원
2) 부분 환불: 금 XXX만원
3) 상품 교환 (선택 시 환불 대신)
4) 반품 배송비 부담 주체 확인 (판매자 하자인 경우 판매자 부담)
```

## Step RF6. 이행 기한
```
권장: 도달 후 3~7일 이내
(전자상거래법 §18: 청약철회 시 3영업일 이내 환불 의무)
```
