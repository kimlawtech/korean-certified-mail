# 인터뷰: 계약 해제·해지 통보 내용증명

## 적용 상황

- 상대방의 계약 불이행으로 계약을 해제하려는 경우
- 임대차·용역·매매 계약 해지 통보
- 최고(催告) 후 해제 또는 즉시 해제

---

## Step T1. 발신인 정보

```
해제(해지)를 통보하는 쪽 정보를 알려주세요.

1) 성함 또는 회사명·대표자명
2) 주소
3) 연락처
```

## Step T2. 수신인 정보

```
상대방 정보를 알려주세요.

1) 성함 또는 회사명·대표자명
2) 주소
3) 연락처 (알고 있는 경우)
```

## Step T3. 계약 개요

```
어떤 계약을 해제(해지)하시려는 건가요?

1) 계약 종류: 매매·임대차·용역·도급·기타
2) 계약일: YYYY년 M월 D일
3) 계약 내용 요약
   예) "2025년 3월, 인테리어 공사 계약 / 공사 금액 5,000만원"
4) 계약서 있음/없음
```

## Step T4. 해제 사유

```
왜 계약을 해제(해지)하려는 건가요?

1) 이행지체 — 상대방이 기한까지 이행하지 않음
   → 이행 기한: YYYY년 M월 D일
   → 미이행 내용:

2) 이행불능 — 상대방이 이행을 못하는 상황
   → 사유:

3) 하자·불완전 이행 — 이행했지만 내용이 불완전
   → 하자 내용:

4) 기타 — 직접 입력
```

## Step T5. 사전 최고 여부

```
이미 상대방에게 이행을 요청한 적 있나요?

1) 있음 — 언제, 어떤 방식으로?
   → 기한을 얼마나 줬나요?
   → 반응은?

2) 없음 — 이번 내용증명으로 최고 + 해제 통보를 같이 합니다.

※ 이행지체로 해제하려면 원칙적으로 "상당한 기간"을 정해 먼저 이행을 최고해야 합니다.
  이번에 처음 요구하는 경우: 기한 내 이행 최고 + 미이행 시 해제 효력 발생으로 작성합니다.
  이미 최고했다면: 즉시 해제 통보로 작성합니다.
```

## Step T6. 원상회복 및 손해배상

```
계약 해제 후 요구사항이 있나요?

1) 원상회복 청구
   - 이미 지급한 대금 반환: 금 XXX만원
   - 반환할 물건:

2) 손해배상 청구
   - 발생한 손해 내역:
   - 금액 (예상):

3) 해당 없음
```

## Step T7. 이행 기한

```
상대방이 이행 또는 원상회복해야 할 기한을 지정하시겠어요?

권장: 내용증명 도달 후 14일 이내
직접 지정: ___일 이내

즉시 해제(해지) 효력 발생: 도달 즉시 (이행불능·즉시 해제 사유인 경우)
```
